# Prueba Técnica — Científico de Datos para Soluciones con IA

Este repositorio contiene la solución completa a la prueba técnica del proceso de selección para el cargo de Científico de Datos con IA en el equipo de Canales Digitales de Protección. El proyecto abarca análisis exploratorio, segmentación de clientes, arquitectura de recomendación en tiempo real, análisis geoespacial y diseño de prompts para un agente de IA conversacional hiper-personalizado.


## Desafíos
| # | Desafío                                   | Técnicas utilizadas                                                                 |
|---|------------------------------------------|--------------------------------------------------------------------------------------|
| 1 | Top 5 productos por volumen e ingresos   | Análisis descriptivo, agregaciones, JOIN entre tablas                                |
| 2 | Mayor dolor de clientes                  | Análisis de texto, frecuencia de palabras clave, análisis de impacto logístico       |
| 3 | Segmentación de clientes                 | Matriz RFM, K-Means, método del codo, StandardScaler                                 |
| 4 | Arquitectura de recomendación en tiempo real | Diseño de sistema híbrido (colaborativo + content-based + RFM)                   |
| 5 | Mejor ubicación para centro comercial    | Análisis geoespacial, agregación por zipcode, geolocalización                        |
| 6 | Base de conocimientos (JSON)             | Extracción de precios reales del dataset, redacción de descripciones                 |
| 7 | System prompts para agente IA            | Diseño de prompts con variables dinámicas para 3 perfiles de cliente                 |
